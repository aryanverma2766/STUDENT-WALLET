# StudentWallet 💸

StudentWallet is a web-based application designed to help students manage their daily expenses in a simple and organized way. The idea behind this project is to make money tracking easier, more visual, and a little smarter with helpful insights.

This project is built using HTML, CSS, JavaScript, and jQuery, and focuses on both functionality and user experience.

---

## About the Project

Managing money as a student can be difficult, especially when small daily expenses go unnoticed. This project helps users keep track of their spending, stay within a budget, and plan savings goals.

The application provides a dashboard where users can clearly see how much they have spent, how much is left from their budget, and where their money is going.

---

## Key Features

- Add and manage daily expenses  
- Categorize expenses (Food, Transport, Books, Fun, etc.)  
- Set a monthly budget and track remaining balance  
- Get visual insights using charts and graphs  
- Create and track savings goals  
- Split bills easily among friends  
- View monthly expense reports  
- Export data as CSV  
- Toggle dark mode for better UI experience  
- Quick add feature for faster expense entry  

---

## Unique Feature

### AI Spend Advisor 🤖

One of the main highlights of this project is the **AI Spend Advisor**.

Instead of just showing numbers, this feature analyzes the user’s spending and provides simple suggestions. It also allows the user to ask questions related to their expenses, like:

- “Where did I spend the most this month?”
- “How can I save more money?”

## Project Structure

The application is divided into different sections:

### 1. Dashboard
Shows an overview of:
- Total spending  
- Remaining budget  
- Daily expenses  
- Number of transactions  
- Weekly spending trend  
- Category-wise breakdown  

### 2. Expenses Section
- Add new expenses  
- Select category  
- Add date and notes  
- Search and filter expenses  
- Delete entries  

### 3. Budget Section
- Set monthly budget  
- Add category limits  
- Track budget usage  

### 4. Savings Goals
- Create savings targets  
- Track progress  
- Set deadlines  
- Visual progress bars  

### 5. Split Bill
- Enter total bill  
- Divide among people  
- Optional custom percentage  
- Add your share as expense  

### 6. AI Advisor
- Analyze spending  
- Provide tips  
- Answer user queries  

### 7. Monthly Report
- View summary of expenses  
- Analyze spending trends
